Question-1: What are the different ways to select an element in the DOM?

In Document Object Model (DOM), there are several ways to select elements. Among them the most prominent ones are mentioned below:
- getElementById()
- getElementsbyClassName()
- getElementsByTagName()
- querySelectorAll()
- querySelector()

Question-2: What is the difference between innerHTML, innerText, and textContent ?

All three of the properties are used to manipulate contents inside tags or a container in the html. Firstly, the innerHTML is different from the other two in the sense that it is used to access the raw html elements along with the text contents by the means of JavaScript. While innerText and textContent work in a more similar fashion. innerText is used to manipulate and access the text contents inside a tag or a html element whereas textContent is used to access not only the text elements but also other hidden elements inside a tag or html content.

Question-3: What is event delegation in the DOM?

Event delegation is the phenomenon by which several child elements are controlled by their parent element. This process utilizes the bubbling propagation of the DOM hierarchy. By this process, the developing is made more efficient.

Question-4: What is event bubbling in the DOM?

Event bubbling the process of bubbling is up of an event from the targeted element to the root of the DOM tree when being triggered. During this process, the bubbling up triggers all the event listeners in it's way up until the event is stopped individually.

Question-5: How do you create, add, and remove elements using JavaScript?

In JavaScript, elements can be created, added and removed dynamically in various ways. An element can be created using the document.createElement() property. While, appendChild() is used to add a child element to a parent block. Similarly, removeChild() property is used to remove element from it's parent element.
